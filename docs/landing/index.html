Phloem

Local-first AI memory with causal graphs

Persistent memory for AI coding tools. Your conversations and context survive across sessions. Fully offline. Zero config. Your data never leaves your machine.

Install with Homebrew View on GitHub
$ brew install canopyhq/tap/phloem

How It Works

More than vector search. Phloem understands how your memories relate to each other and to your code.

Causal Memory Graphs

Not just vector search. Phloem builds a directed acyclic graph linking memories by cause and effect.

Citation Verification

Memories linked to specific code locations. Confidence scores decay when code changes.

Works With Your IDE

MCP protocol support for Claude Code, Cursor, and Windsurf. One command setup.

Your Data Stays on Your Machine

No network calls. No accounts. No telemetry. Everything lives in a local SQLite database.

  • + All memories stored locally in ~/.phloem/memories.db
  • + Zero network calls -- works fully offline
  • + No account required, no sign-up
  • + No telemetry, no analytics, no tracking
  • + Open source -- audit the code yourself

Quick Start

Two commands. Under a minute.

# Install via Homebrew
brew install canopyhq/tap/phloem

# Set up Phloem
phloem setup

IDE Support

Connect Phloem to your AI coding tool with a single command.

Claude Code
phloem setup claude-code
Cursor
phloem setup cursor
Windsurf
phloem setup windsurf
